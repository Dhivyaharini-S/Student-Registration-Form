# Student-Registration-Form
A C# Windows Forms application with photo upload, live camera capture, personal details form, validation, reset, print preview, and PDF export functionality.
## Features

- Upload photo from computer
- Open laptop or USB camera
- Capture current photo
- Enter personal details
- Select gender and marital status
- Select qualifications
- Choose experience level
- Add project details
- Reset the form
- Submit and display entered details
- Print preview
- Save registration details as PDF

## Technologies Used

- C#
- Windows Forms
- .NET Framework
- AForge.Video
- AForge.Video.DirectShow
- Visual Studio

## How to Run

1. Clone or download the repository.
2. Open the solution file in Visual Studio.
3. Restore the required NuGet packages.
4. Build the solution.
5. Press `F5` to run the application.
6. Allow camera permission in Windows settings.
